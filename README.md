### 세부 커리큘럼 및 폴더 매핑

#### 1. JAVA (자바 프로그래밍) (2025.10.23 ~ 2025.11.08)

JAVA 기초 문법부터 객체지향 프로그래밍(OOP)의 핵심 개념을 다룹니다.

- **기초 및 제어문**: `JAVA/Java20251022`, `JAVA/Java20251023`
- **상속 (Inheritance)**: `JAVA/Java20251031_상속`
- **다형성 (Polymorphism)**: `JAVA/Java20251103_다형성`, `JAVA/Java20251104_다형성`
- **인터페이스 (Interface)**: `JAVA/Java20251105_인터페이스`
- **예외처리 (Exception Handling)**: `JAVA/Java20251106_예외처리`
- **제네릭 (Generics)**: `JAVA/Java20251106제네릭`
- **컬렉션 프레임워크 (Collections)**: `JAVA/Java20251107_컬렉션`

#### 2. HTML / CSS (웹 표준) (2025.10.23 ~ 2025.11.06)

웹 페이지 구조와 스타일링의 기본을 학습합니다.

- **HTML**: 텍스트, 이미지, 링크, 폼 요소 등 (`HTML/2_text태그` ~ `HTML/4_form`)
- **CSS**: 스타일링 기초, 레이아웃, 리셋 (`HTML/14_reset_...` 등`)

#### 3. JavaScript (자바스크립트) (2025.11.10 ~ 2025.11.18)

클라이언트 사이드 스크립팅 언어를 학습합니다.

- **기초 문법**: 변수, 연산자, 함수 등 (`JS/1_첫째마당 소스_Js` 등)
- **제어문 및 객체**: `JS/4장`, `JS/5장`, `JS/6장`
- **비동기 처리**: `JS/callBack.js`, `JS/callBack_examples.js`

#### 4. JSP / Servlet (웹 프로그래밍 기초) (2025.11.10 ~ 2025.11.27)

서버 사이드 웹 프로그래밍의 기초를 다룹니다.

- **기초 실습**: `JSP/web-study-01` ~ `JSP/web-study-11`

#### 5. Database (데이터베이스) (2025.11.14 ~ 2025.11.18)

- **SQL 및 연동**: `SQL` 디렉토리 (MySQL, Oracle 등 JDBC 연동 실습 포함)

#### 6. 대철 프로젝트 (Daecheol Project) (2025.11.20 ~ 2025.11.25)

별도 저장소에서 진행된 JSP/Servlet 기반 프로젝트입니다.

- **내용**: 게시판 관련 프로젝트 (`board-project`)
- **기술 스택**: Java 21, JSP, Servlet, JSTL, Apache Tomcat 9.0
- **위치**: `../daechul-project/board-project` (외부 경로)
- **저장소**: [https://github.com/every-git/daechul-project](https://github.com/every-git/daechul-project)

#### 7. React (리액트) (2025.11.27 ~ )

컴포넌트 기반의 UI 라이브러리 학습 및 프로젝트 진행.

- **기초 및 훅(Hooks)**: `React/chapter4`, `React/chapter5`
- **미니 프로젝트**: `React/project1`
- **심화 및 응용 프로젝트**:

  - **다양한 프로젝트 예제**: `React_1120/project1` ~ `React_1120/project3`

엔터프라이즈급 애플리케이션 개발을 위한 프레임워크를 학습합니다.
