# 🎯 Connect to Server 다이얼로그 - 입력 가이드 (간단 버전)

화면에 보이는 순서대로 입력하세요!

---

## 📝 연결 1: Local instance 3306

### 맨 위 부분 (연결 정보)
```
┌─────────────────────────────────────┐
│ Name: Local instance 3306           │ ← 연결 구분할 이름 (원하는 대로 입력)
│ Group:                              │ ← 비워두기 (선택사항, 안 써도 됨)
│ [Advanced] [Scope: Global]          │ ← 그대로 두기
└─────────────────────────────────────┘
```

### 서버 타입
```
MySQL (이미 선택되어 있음) ← 그대로 두기
```

### 메인 설정 부분
```
┌─────────────────────────────────────┐
│ * Host: localhost                   │ ← 여기에 "localhost" 입력
│ * Port: 3306                        │ ← 그대로 3306 (또는 -/+ 버튼으로 조정)
│ * Username: root                    │ ← 여기에 "root" 입력
│ * Password: [비밀번호 입력]          │ ← root 비밀번호 입력
│ Database: dcproject                 │ ← 여기에 "dcproject" 입력 (선택사항)
│ SSL: [ ]                            │ ← 체크박스 해제 (SSL 끄기)
└─────────────────────────────────────┘
```

**✅ 입력 요약:**
- Name: `Local instance 3306`
- Group: (비워두기)
- Host: `localhost`
- Port: `3306`
- Username: `root`
- Password: (실제 비밀번호)
- Database: `dcproject`
- SSL: ❌ 끄기

## 🤔 Name과 Group은 뭐하는 건가요?

### **Name (이름)**
- **용도**: 이 연결을 나중에 찾을 때 구분하는 이름
- **입력**: 원하는 이름 아무거나 입력
  - 예: `Local instance 3306`, `jdbctest`, `NAS_DB`, `springdbuser`
- **필수**: 예 (별표 * 표시됨)

### **Group (그룹)**
- **용도**: 여러 연결을 묶어서 관리할 때 사용
- **입력**: 
  - 선택사항 - **비워둬도 됩니다!**
  - 원하면 `Local`, `Remote`, `Development` 같은 단어 입력
- **필수**: 아니요 (선택사항)

**💡 간단히 말하면:**
- **Name**: 연결 이름 (필수) - "어떤 연결인지 알려주는 이름"
- **Group**: 그룹 이름 (선택) - "비워둬도 됨"

---

## ⚠️ 주의할 점 3가지

1. **Local instance 3306**만 SSL 끄기 (나머지는 모두 SSL 켜기)
2. **NAS_DB**는 Port가 `3307` (나머지는 `3306`)
3. **springdbuser**는 Database가 `springdb` (나머지는 `dcproject`)
