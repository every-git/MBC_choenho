# 비밀번호 설정 위치 및 업데이트 방법

## 📍 비밀번호를 넣어야 하는 파일 위치

**파일 경로:**
```
~/Library/Application Support/Cursor/User/globalStorage/cweijan.vscode-database-client2/connections.json
```

방금 이 파일을 열었습니다! 

---

## ✏️ 비밀번호 입력 방법

### 파일이 열렸다면:

각 연결의 `password` 필드를 찾아서 실제 비밀번호로 수정하세요:

```json
{
  "name": "springdbuser",
  "host": "127.0.0.1",
  "port": 3306,
  "username": "root",
  "password": "여기에_실제_비밀번호_입력",  // ← "1234"를 실제 비밀번호로 변경
  "database": "springdb",
  ...
}
```

### root 계정 사용 연결들:
- **Local instance 3306** (7번 줄의 password)
- **springdbuser** (43번 줄의 password)

둘 다 root 비밀번호를 사용하므로 같은 값으로 설정하세요.

---

## 🔍 현재 설정 확인

현재 파일에는 `"password": "1234"`로 설정되어 있습니다.

터미널에서 테스트한 비밀번호가 "1234"가 아니라면, 파일에서 "1234"를 실제 비밀번호로 변경해야 합니다.

---

## ✅ 변경 후 할 일

1. 파일 저장 (Cmd + S)
2. Cursor 재시작 (필수!)
3. Database Client에서 연결 테스트

---

## 💡 빠른 찾기

파일에서 `"password":` 를 검색하면 모든 비밀번호 필드를 빠르게 찾을 수 있습니다.
